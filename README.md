# PDV-Carrinho-de-praia
Overview
O PDV para Carrinhos de Praia resolve problemas comuns no atendimento de clientes na areia, como a demora no atendimento, a falta de atualização dos pedidos e a dificuldade de comunicação, já que muitas vezes os clientes estão distantes do carrinho.
Com o app, os clientes podem visualizar o cardápio, fazer pedidos diretamente pelo celular e acompanhar o status em tempo real (Pendente, Preparando, Entregue). O operador do carrinho recebe os pedidos organizados por número de guarda-sol ou mesa, agilizando o atendimento e reduzindo erros.
